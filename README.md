# Leetcode-practice

LeetCode 练习仓库 | LeetCode Practice Repository

## 简介 | Introduction

这是一个用于记录和分享 LeetCode 刷题解答的仓库。

This repository is used to record and share LeetCode problem solutions.

## 目录结构 | Directory Structure

```
Leetcode-practice/
├── README.md
├── LICENSE
└── .gitignore
```

## 使用方法 | Usage

1. 克隆仓库 | Clone the repository
   ```bash
   git clone https://github.com/NiceAsiv/Leetcode-practice.git
   ```

2. 添加你的解题代码 | Add your solution code

3. 编译运行 | Compile and run
   
   C++:
   ```bash
   g++ -o solution solution.cpp && ./solution
   ```
   
   Python:
   ```bash
   python solution.py
   ```
